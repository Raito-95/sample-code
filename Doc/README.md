# 文件總覽

## 目的

`Doc/` 是專案文件入口，僅提供分類導覽，不重複各主題細節。

## 內容清單

- `Doc/Tools/`：工具需求與使用文件
- `Doc/Complexity/Complexity.md`：時間與空間複雜度
- `Doc/DesignPatterns/README.md`：設計模式索引
- `Doc/Python/OOP_Basics.md`：Python 物件導向基礎
- `Doc/Python/OOP_Advanced.md`：Python 物件導向進階

## 建議閱讀順序

1. 先讀 `Doc/README.md` 了解全貌。
2. 再依需求進入對應主題或子資料夾。
3. 細節以各主題文件為準。

## 維護規則

1. 本檔只放導覽與規則，不放教學細節。
2. 新增/更名文件時，必須同步更新本索引。
3. 術語請與子文件保持一致。
