# student_grade_manager

- 路徑: `apps/algorithms_lab/student_grade_manager.py`
- 狀態: `active`

## 工具目的

提供 CLI 學生成績管理流程，支援查詢、新增、刪除與資料持久化。

## 功能需求 (FR)

- `FR-1`: 讀取文字檔成績資料並載入記憶體。
- `FR-2`: 查詢單一學生單科成績。
- `FR-3`: 查詢單一學生全部科目成績。
- `FR-4`: 新增或覆寫學生完整成績資料。
- `FR-5`: 刪除學生成績資料。
- `FR-6`: 結束時將資料寫回檔案。

## 非功能需求 (NFR)

- `NFR-1`: 對格式錯誤輸入要明確拋出錯誤訊息。
- `NFR-2`: 基本邏輯可由單元測試驗證。
- `NFR-3`: 優先維持簡單、可教學閱讀的實作。

## 已知限制

- 目前為互動式 CLI，未提供 GUI。
- 資料檔格式固定為：`student_id subject score ...`。
