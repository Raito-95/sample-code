# 設計模式索引

## 目的

本資料夾整理 GoF 設計模式，提供分類導覽與閱讀入口。

## 內容清單

- `CreationalPatterns/`（建立型）
- `StructuralPatterns/`（結構型）
- `BehavioralPatterns/`（行為型）

建立型：`BuilderPattern.md`、`FactoryMethod.md`、`PrototypePattern.md`、`SingletonPattern.md`

結構型：`AdapterPattern.md`、`DecoratorPattern.md`、`ProxyPattern.md`

行為型：`CommandPattern.md`、`ObserverPattern.md`、`StrategyPattern.md`

## 建議閱讀順序

1. 先看各模式的「概觀」。
2. 再看「Python 實作」理解角色關係。
3. 最後執行「程式範例」對照輸出。

## 維護規則

1. 本檔只維護分類與導覽，不重複各模式內容。
2. 新增或移除模式文件時，需同步更新本索引。
3. 標題命名與術語請在同類型文件中保持一致。
